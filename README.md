Sistema simples de cadastro de pessoas em Java (nome, idade e altura)

# 💻 Cadastro de Pessoas em Java
Este projeto foi desenvolvido como parte dos meus estudos em Java, com foco em Programação Orientada a Objetos.

## 🚀 Funcionalidades
- Cadastro de uma pessoa
- Exibição de dados (nome, idade e altura)

## 🧠 Tecnologias utilizadas
- Java

## 📚 Conceitos aplicados
- Classes e objetos
- Construtores
- Métodos

## 🎯 Objetivo
Praticar lógica de programação e desenvolver habilidades em Java durante minha transição para a área de tecnologia.
